# 测试方法：CIE色度坐标计算

## 目的

将电致变色材料不同价态（不同电压）下的吸收光谱转换为CIE 1931色度坐标，定量表征电致变色过程中的颜色变化。

## 适用范围

电致变色材料（WO₃、TiO₂@PBI、PBI等）在不同氧化还原态下的颜色表征。

## 所需仪器与工具

| 仪器/软件 | 用途 |
|-----------|------|
| 紫外-可见分光光度计 | 获取不同价态样品的吸收/透射光谱 |
| www.corlortell.com | 将反射率数据转换为XYZ三刺激值 |
| Origin | 绘制CIE 1931色度图 |

## 操作步骤

### 第一步：获取吸收光谱
1. 使用紫外-可见分光光度计分别测量不同价态（施加不同电压）下样品的吸收光谱或透射光谱。
2. 光谱范围至少覆盖380~780 nm（可见光区），步长建议1 nm或5 nm。
3. 记录每个价态对应的波长-吸光度（或透过率）数据。

### 第二步：转换为反射数据
1. 若测量的是透射光谱T(λ)，对于不透明薄膜可近似认为反射率 R(λ) ≈ 1 − T(λ) − A(λ) 中的适用部分；或直接用吸收光谱反推光谱反射特性。
2. 若样品可直接测量反射光谱，则直接使用反射数据。
3. 整理成两列数据：波长(nm)、反射率R(λ)（取值0~1或0~100）。

### 第三步：XYZ三刺激值转换
1. 打开网站 www.corlortell.com（注意拼写，实际网址建议搜索确认）。
2. 在网站中选择CIE标准色度观察者（CIE 1931 2° Standard Observer）。
3. 选择光源类型：通常选用D65（标准日光）或A光源。
4. 输入或粘贴反射光谱数据。
5. 网站计算输出XYZ三刺激值。
6. 记录每个价态对应的X、Y、Z值。

### 第四步：转换为Yxy色坐标
1. 由XYZ三刺激值计算色坐标(x, y)：
   - x = X / (X + Y + Z)
   - y = Y / (X + Y + Z)
   - Y保持不变（代表亮度信息）
2. 可在Excel或Origin中完成此计算。

### 第五步：绘制色度图
1. 打开Origin，导入各价态的(x, y)数据。
2. 在Origin中绘制CIE 1931色度图模板（Origin内置或自行导入CIE马蹄形边界数据）。
3. 将各价态的色坐标点标注在色度图上。
4. 不同价态以不同颜色/符号区分，可用箭头标注颜色变化路径。
5. 色度图选择CIE 1931标准。

## 数据分析

1. 观察不同价态色坐标在马蹄形图中的位置变化，判断颜色变化范围和趋势。
2. 计算相邻价态之间的色差 ΔE（可选）。
3. 分析颜色变化效率——单位电荷注入引起的色坐标位移量。

## 关键参数

| 参数 | 建议设置 | 说明 |
|------|----------|------|
| 标准观察者 | CIE 1931 2° | 小视场角，适用于薄膜样品 |
| 光源 | D65 | 模拟自然日光 |
| 波长范围 | 380~780 nm | 可见光全波段 |
| 波长步长 | 5 nm | 步长越细精度越高 |

## 注意事项

- 吸收光谱的基线需校正，避免基线漂移影响颜色计算准确性。
- 反射率数据需归一化至0~1（或0~100 %），输入网站时注意格式要求。
- 不同光源选择会影响XYZ值，全文应统一光源。
- CIE 1931色度图为非均匀色空间，色坐标的几何距离不等于感知色差（如需感知均匀空间可使用CIE 1976 L*a*b*）。
- 薄膜样品的反射光谱受基底影响，尽量使用积分球附件测量漫反射光谱。
- 网址 www.corlortell.com 可能拼写有误，实际使用时请确认正确域名（常见工具有：ColorTell、EasyRGB等）。
